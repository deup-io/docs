# BlockType

块类型, 枚举类型.

```typescript
enum BlockType {
  PAGE = 'page',
  TEXT = 'text',
  TEXT_LIST = 'text/list',
  IMAGE = 'image',
  VIDEO = 'video',
}
```

### page

页面类型, `BlockType.PAGE`

### text

文本类型, `BlockType.TEXT`

### text/list

文本列表类型, `BlockType.TEXT_LIST`

### image

图片类型, `BlockType.IMAGE`

### video

视频类型, `BlockType.VIDEO`
