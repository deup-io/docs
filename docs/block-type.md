# BlockType

块类型, 枚举类型.

```typescript
enum BlockType {
  PAGE = 'page',
  TEXT = 'text',
  RICH_TEXT = 'richText',
  IMAGE = 'image',
  VIDEO = 'video',
}
```

### page

页面类型, `BlockType.PAGE`

### text

文本类型, `BlockType.TEXT`

### richText

富文本类型, `BlockType.RICH_TEXT`

### image

图片类型, `BlockType.IMAGE`

### video

视频类型, `BlockType.VIDEO`
